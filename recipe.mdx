# Setup Gatsby with Preact

[Preact](https://preactjs.com/) is a leaner React alternative that you can enable to make your Gatsby sites faster

---

Install necessary NPM packages

<NPMPackage name="gatsby-plugin-preact" />
<NPMPackage name="preact" />

---

Install the Preact plugin in gatsby-config.js

<GatsbyPlugin name="gatsby-plugin-preact" />

---
